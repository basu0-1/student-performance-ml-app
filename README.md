## 📌 Overview

The **Student Performance Predictor** is a Machine Learning-based web application that predicts a student's final grade based on study habits and personal attributes.

This project demonstrates how a machine learning model can be transformed into a real-world interactive application with user input, prediction logic, and data tracking.

---

## ✨ Key Features

- 🔐 Simple user login system  
- 📊 Real-time student grade prediction  
- 💾 Stores prediction history in CSV  
- 📝 Tracks user login/logout activity  
- 🧠 ML-powered decision making  
- 🖥️ Clean and interactive interface  

---

## 🧠 Machine Learning Model

- **Algorithm Used:** Random Forest Regressor  
- **Problem Type:** Regression  
- **Target Variable:** Final Student Grade  

### 📥 Input Features:
- Age  
- Study Time  
- Absences  
- Gender  
- School  

The model is trained on student performance data and predicts outcomes based on behavioral patterns.

## 📊 How It Works

1. User enters the application through a login interface  
2. Inputs student-related details  
3. Machine learning model processes the data  
4. Predicts the final grade  
5. Saves prediction and logs user activity
